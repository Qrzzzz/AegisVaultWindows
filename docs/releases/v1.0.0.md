# AegisVault v1.0.0

Release date: 2026-06-28

AegisVault 1.0.0 is the first stable public release of the local Windows desktop encryption utility. It provides modern text and file encryption, Base64 encode/decode tools, and explicit recovery paths for supported legacy data.

## Downloads

- `AegisVault-v1.0.0-win64.zip` - standalone Windows x64 build. Extract the ZIP and run `AegisVault.exe`.

## Security Scope

- Modern text encryption writes `AGV1.` tokens.
- Modern file encryption writes chunked `.agv` containers.
- New encryption uses scrypt and AES-256-GCM with per-encryption random salt.
- Legacy recovery is migration-only.
- AK wrapper parsing is disabled by default because AK data embeds key material.

AegisVault data is locally encrypted with documented design and tests, but not independently audited.

## Compatibility

- Existing `AGV1.` text tokens remain supported.
- Existing `.agv` file containers remain supported.
- Legacy AES Encryption System data can be recovered for migration when the user has the original password.

## Verification

The release verification command is:

```powershell
.\scripts\verify_release.ps1 -Build -Zip
```

That command compiles source and tests, runs Ruff, mypy, pytest, a headless application smoke test, builds the Windows executable, and creates `dist\AegisVault-v1.0.0-win64.zip`.

## Known Limitations

AegisVault does not protect against malware, clipboard monitoring, screenshots, screen recording, weak passwords, plaintext backups, compromised machines, or lost passwords. File names and filesystem metadata may still reveal sensitive context.
